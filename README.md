# TaxFlow ES

**TaxFlow ES** es una aplicación de contabilidad con IA para autónomos y pymes en Espańa.

Extrae datos de facturas, tickets y recibos (NIF, IVA, importes, etc.), organiza gastos e ingresos y facilita la preparación de modelos fiscales como el 303 y 130.  
No sustituye el asesoramiento fiscal profesional.

## Características principales
- Análisis inteligente de documentos usando LLMs
- Soporte para múltiples proveedores de IA (OpenAI, Groq, Mistral, Google, etc.)
- Totalmente self-hosted y privado
- Interfaz moderna
- Soporte Docker

## Tecnologías
- Next.js
- Prisma
- LLM integration
- Docker Compose
